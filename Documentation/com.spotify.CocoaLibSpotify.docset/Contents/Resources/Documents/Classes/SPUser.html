Overview

Represents a user of the Spotify service.

SPUser is roughly analogous to the sp_user struct in the C LibSpotify API.

Class Methods

userWithURL:inSession:

Creates an SPUser from the given Spotify user URL.

+ (SPUser *)userWithURL:(NSURL *)userUrl inSession:(SPSession *)aSession

Parameters

userUrl

The user URL to create an SPUser for.

aSession

The SPSession the user should exist in.

Return Value

Returns the created SPUser object, or nil if given an invalid user URL.

Discussion

This convenience method creates an SPUser object if one doesn’t exist, or returns a cached SPUser if one already exists for the given URL.

Important: If you pass in an invalid user URL (i.e., any URL not starting spotify:user:, this method will return nil.

relationshipWithUser:

Returns the relationship of the user with another user.

- (sp_relation_type)relationshipWithUser:(SPUser *)friendOrFoe

Parameters

friendOrFoe

The user to get the relationship status of.

Discussion

Possible values are:

SP_RELATION_TYPE_UNKNOWN
Not yet known.

SP_RELATION_TYPE_NONE
No relation.

SP_RELATION_TYPE_UNIDIRECTIONAL
The currently logged in user is following this uer.

SP_RELATION_TYPE_BIDIRECTIONAL
Bidirectional friendship established.
