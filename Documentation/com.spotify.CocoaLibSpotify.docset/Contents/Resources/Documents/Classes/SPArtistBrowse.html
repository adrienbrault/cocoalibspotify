Overview

Represents an “artist browse” of an artist on the Spotify service.

An “artist browse” fetches detailed information about an artist from the Spotify service, including a biography, portrait images, related artists and a list of their tracks and albums.

Artist or album browses are required for certain SPTrack metadata to be available – see the SPTrack documentation for details.

browseArtistAtURL:inSession:

Creates an SPArtistBrowse from the given artist URL.

+ (SPArtistBrowse *)browseArtistAtURL:(NSURL *)artistURL inSession:(SPSession *)aSession

Parameters

artistURL

The artist URL to make an SPArtistBrowse for.

aSession

The SPSession the browse should exist in.

Return Value

Returns the created SPArtistBrowse object.

Discussion

This convenience method is simply returns a new, autoreleased SPArtistBrowse object. No caching is performed.

Important: If you pass in an invalid artist URL (i.e., any URL not starting spotify:artist:, this method will return nil.
