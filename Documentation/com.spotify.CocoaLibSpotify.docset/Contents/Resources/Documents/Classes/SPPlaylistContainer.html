Overview

This class represents a list of playlists. In practice, it is only found when dealing with a user’s playlist list and can’t be created manually.

playlists

Returns an array of SPPlaylist and/or SPPlaylistFolders representing the owner’s playlists.

@property (readonly) NSMutableArray *playlists

Discussion

This array is KVO compliant, and any changes made will be reflected in the user’s account.

Important: If you need to move a playlist from one location in this list to another, please use movePlaylistOrFolderAtIndex:ofParent:toIndex:ofNewParent:error: for performance reasons.

movePlaylistOrFolderAtIndex:ofParent:toIndex:ofNewParent:error:

Move a playlist or folder to another location in the list.

- (BOOL)movePlaylistOrFolderAtIndex:(NSUInteger)aPlaylistOrFolder ofParent:(SPPlaylistFolder *)existingParentFolderOrNil toIndex:(NSUInteger)newIndex ofNewParent:(SPPlaylistFolder *)aParentFolderOrNil error:(NSError **)err

Parameters

aPlaylistOrFolder

The index of the playlist or folder in its parent (or the root list if it has no parent).

existingParentFolderOrNil

The parent folder the previous index refers to, or nil if there is no parent.

newIndex

The desired destination index in the destination parent folder (or root list if there’s no parent).

aParentFolderOrNil

The new parent folder, or nil if there is no parent.

err

An NSError pointer to be filled if the operation fails.

Return Value

Returns YES if the operation succeeded, otherwise NO.

Discussion

Important: This operation can fail, for example if you give invalid indexes or try to move a folder into itself. Please make sure you check the result of this method.
