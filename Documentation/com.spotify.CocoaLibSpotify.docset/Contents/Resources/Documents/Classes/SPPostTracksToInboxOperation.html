Class Methods

sendTracks:toUser:message:inSession:delegate:

Creates an SPPostTracksToInboxOperation for the given details.

+ (SPPostTracksToInboxOperation *)sendTracks:(NSArray *)tracksToSend toUser:(NSString *)user message:(NSString *)aFriendlyGreeting inSession:(SPSession *)aSession delegate:(id<SPPostTracksToInboxOperationDelegate>)completionDelegate

Parameters

tracksToSend

An array of SPTrack objects to send.

user

The username of the user to send the tracks to.

aFriendlyGreeting

The message to send with the tracks, if any.

aSession

The session to send the tracks with.

completionDelegate

The delegate to send success/failure messages to.

Return Value

Returns the created SPPostTracksToInboxOperation object.

Discussion

This convenience method is simply returns a new, autoreleased SPPostTracksToInboxOperation object. No caching is performed.

Important: Tracks will be posted to the given user as soon as a SPPostTracksToInboxOperation object is created. Be sure you want to post the tracks before creating the object!
