folderId

Returns the folder’s ID, as used in the C LibSpotify API.

@property (readonly) sp_uint64 folderId

Discussion

Important: This should only be used if you plan to directly use the C LibSpotify API. The behaviour of CocoaLibSpotify is undefined if you use the C API directly on items that have CocoaLibSpotify objects associated with them.

playlists

Returns an array of SPPlaylist and/or SPPlaylistFolders representing the folder’s child playlists.

@property (readonly) NSMutableArray *playlists

Discussion

This array is KVO compliant, and any changes made will be reflected in the user’s account.

Important: If you need to move a playlist from one location in this list to another, please use [SPPlaylistContainer movePlaylistOrFolderAtIndex:ofParent:toIndex:ofNewParent:error:] for performance reasons.
