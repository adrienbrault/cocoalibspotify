Overview

Represents a track on Spotify.

SPTrack is roughly analogous to the sp_track struct in the C LibSpotify API.

consolidatedArtists

Returns a string represention of the artist(s) of the track. If no metadata is available for the track yet, returns nil.

@property (readonly, copy) NSString *consolidatedArtists

Discussion

If the track has one artist, returns that artist’s name. Otherwise, returns all artist names in alphabetical order, each separated with a comma (,).

Returns the disc index of the track.

@property (readonly) NSUInteger discNumber

trackForTrackURL:inSession:

Creates an SPTrack from the given Spotify track URL.

+ (SPTrack *)trackForTrackURL:(NSURL *)trackURL inSession:(SPSession *)aSession

Parameters

trackURL

The track URL to create an SPTrack for.

aSession

The SPSession the track should exist in.

Return Value

Returns the created SPTrack object, or nil if given an invalid track URL.

Discussion

This convenience method creates an SPTrack object if one doesn’t exist, or returns a cached SPTrack if one already exists for the given URL.

Important: If you pass in an invalid track URL (i.e., any URL not starting spotify:track:, this method will return nil.
